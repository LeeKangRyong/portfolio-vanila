# 포트폴리오 using VanilaJS

### 구현 목적
기존 **React+Vite, CSS**로 만든 포트폴리오 웹사이트는 그저 포트폴리오 용으로 만들었을 뿐더러, 최적화 등 아무 것도 적용하지 않았다.  
이번 포트폴리오는 보기에는 똑같지만, 오로지 **HTML+CSS+JS**로만 구현하며, **우아한테크코스 프리코스에서 배운 점**을 적용해보려 한다.

        - indent depth < 3

        - 함수 하나에 15줄 이하 + 한 가지 일만 하도록 최대한 작게 만들기

        - commit convention 지키기
            1. AngularJS 적용
            2. 영문으로 적기
            3. 제목은 최대한 50자 이하로. 힘들면 69자까지
